# Netflix Data Analysis

* **Objective:** Cleaned and analyzed the Netflix Movies and TV Shows dataset.
* **Scale:** Processed over 8,800+ rows of data.
* **Steps Taken:**
    * Handled missing values (NaN) in `director`, `cast`, and `country`.
    * Converted `date_added` to proper Datetime format.
    * Created visual trends using Matplotlib.
* **Tools:** Python, Pandas, Matplotlib.
